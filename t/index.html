colin.party/t is a webpage for displaying, copying, and downloading text to a file. Simply type the text after /t/ to display it, like this:

colin.party/t/hello!

the "n" URL parameter can be used to display a note:

colin.party/t/hello!?n=Text below

and the "f" URL parameter can be used to set a file name when downloading:

colin.party/t/hello!?n=Text below&f=filename
